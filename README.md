# Brazilian-E-Commerce-Data-Analysis-Project

## Project Overview
This project analyzes the Brazilian Olist E-Commerce dataset using **SQL** and **Power BI** to uncover insights about sales performance, customer behavior, payments, and delivery efficiency.

## Tools Used
- SQL (BigQuery / MySQL)
- Power BI
- GitHub
- Kaggle Dataset

## Dataset
Brazilian E-Commerce Public Dataset by Olist  
~100,000 orders with multiple tables.

## Business Questions Answered

### Sales Analysis
- Monthly Revenue Trend
- Top Product Categories
- Revenue Growth

### Customer Analysis
- Repeat vs One-Time Customers
- Average Order Value
- Top 10 Customers by Lifetime Value

### Payment Analysis
- Payment Method Usage
- Installments vs Average Order Value

### Delivery Performance
- Average Delivery Time
- Late Delivery Percentage
- Seller Performance Impact

## Key Insights
- Majority of customers are one-time buyers.
- Credit card is the dominant payment method.
- Higher installments slightly increase average order value.
- Delivery delays impact customer reviews.
